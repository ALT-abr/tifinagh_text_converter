## ⵣ Tifinagh_Text_Converter    
### ✨ Convertisseur Latin ⇄ Tifinagh (alphabet amazigh/berbère)

Petit programme Python qui transforme un texte écrit en alphabet latin vers l’alphabet Tifinagh, avec une version pensée pour être simple, rapide et utilisable en ligne de commande.

## 🌟 Fonctionnalités
- Conversion Latin ⇄ Tifinagh
- Suppression automatique des accents (é → e, à → a, ç → c…)
- Utilisation simple dans le terminal
- Code modulaire et facile à étendre

## ⚙️ How it Works
Le programme :
1. Lit l’entrée utilisateur depuis le terminal
2. Nettoie le texte (accents → lettres simples)
3. Parcourt chaque caractère
4. Cherche la correspondance dans un dictionnaire (mapping)
5. Remplace et affiche le résultat en Tifinagh

### Example mapping:
- a → ⴰ
- b → ⴱ
- d → ⴷ
- m → ⵎ
- z → ⵣ
